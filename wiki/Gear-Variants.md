# Gear Variants
Adding the a variant prefix to the beginning of the gear's name adds the listed effect.

## Implemented Variants
### Charging
Adds Power Up to the user

**Available on**
- all Bucklers (Earth)

### Devoted
Can apply to allies

**Available on**
- Bucker (Earth)

### Guarding
Adds Block to the user

**Available on**
- Buckler (Earth)

### Heavy
Increased Block

**Available on**
- Buckler (Earth)
