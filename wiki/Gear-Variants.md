# Gear Variants
Adding the a variant prefix to the beginning of the gear's name adds the listed effect.

## Implemented Variants

### Accelerating
Adds Quicken to the user

**Available on**
- Cloak (Wind)
- Lance (Earth)
- Shortsword (Fire)

### Charging
Adds Power Up to the user

**Available on**
- all Bucklers (Earth)
- Blood Aegis (Darkness)

### Corrosive
Adds Power Down to target

**Available on**
- all Corrosions (Fire)

### Devoted
Can apply to allies

**Available on**
- Bucker (Earth)

### Evasive
Adds Evade to the user

**Available on**
- all Cloaks (Wind)

### Fate-Sealing
Adds Stasis to target on crit

**Available on**
- Censer (Fire)

### Flanking
Adds Exposed to target

**Available on**
- Life Drain (Darkness)
- Corrosion (Fire)

### Guarding
Adds Block to the user

**Available on**
- Buckler (Earth)
- Potion Kit (Water)

### Healing
Heal the user

**Available on**
- all Life Drains (Darkness)

### Heavy
Increased Block

**Available on**
- Buckler (Earth)
- Blood Aegis (Darkness)
- Scutum (Fire)

### Hunter's
Gain gold on kill

**Available on**
- Sickle (Water)

### Lethal
Increased crit damage

**Available on**
- all Daggers (Wind)

### Long
Increased modifier stacks

**Available on**
- Cloak (Wind)

### Organic
Regains durability when entering new rooms

**Available on**
- Iron Fist Stance (Light)
- Potion Kit (Water)

### Piercing
Deals Unblockable damage

**Available on**
- Lance (Earth)

### Reactive
Bonus damage if user moves after target

**Available on**
- Life Drain (Darkness)

### Sharpened
Increased damage

**Available on**
- Daggers (Wind)
- Sickle (Water)

### Slowing
Adds Slow to target

**Available on**
- Daggers (Wind)

### Soothing
Adds Regen to target

**Available on**
- Floating Mist Stance (Light)

### Sweeping
Targets whole team

**Available on**
- Daggers (Wind)
- Blood Aegis (Darkness)
- Scutum (Fire)

### Taunting
Change target's move's target to user

**Available on**
- all Blood Aegis (Darkness)

### Thick
Increased durability

**Available on**
- Cloak (Wind)
- Censer (Fire)

### Tormenting
Increment a stack on all target's debuffs

**Available on**
- Censer (Fire)

### Toxic
Adds Poison to target

**Available on**
- Sickle (Water)
- Shortsword (Fire)

### Urgent
Increased priority

**Available on**
- Potion Kit (Water)
- Life Drain (Darkness)

### Vexing
Bonus damage if target has debuffs

**Available on**
- all Censers (Fire)

### Vigilant
Adds Vigilance to user

**Available on**
- Lance (Earth)
- Scutum (Fire)

## Unnamed Variants
- Adds Floating Mist Stance to user (all Floating Mist Stances)
- Adds Iron Fist Stance to user (all Iron Fist Stances)
- Double benefits from Power Up (all Lances)
- Percent max hp damage (all Sickles)
